# flaskdemo
flask demo

## Architecture
 - FlaskAPP (http://127.0.0.1:8888) - MySQL8.0 (3306)

## Requirements
~~~
 pip3 install flask 
 pip3 install pymysql
 pip3 install pyopenssl
 pip3.8 install flask_wtf
 pip3.8 install email_validator
~~~
만약 pyopenssl 에러 발생시 그대로 진행 해도 무관


## Quick start
~~~
python run.py
~~~
