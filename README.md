# flaskdemo
flask demo

## Architecture
 - FlaskAPP (http://127.0.0.1:8888) - MySQL8.0 (3306)

## Requirements
~~~
 pip3 install flask 
 pip3 install pymysql
 pip3 install pyopenssl
~~~


## Quick start
~~~
python run.py
~~~
